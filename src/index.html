🧠 La Columna Vertebral del Framework

🔌 Módulo del Sistema (_main/)

Toda la inteligencia lógica está centralizada en módulos aislados en src/_main/:

  • constants.js: Credenciales del entorno y URLs dinámicas de red.
  • router.js: Motor de ruteo dinámico sin recarga física.
  • auth.js: Gestión de sesión y doble factor por Whitelist.
  • storage.js: Persistencia local y remota en Realtime Database.
  • i18n.js: Soporte de traducción y localización dinámica.
  • analytics.js: Telemetría y taxonomía unificada de eventos.
  • util.js: Caja de herramientas estructurales y viewport móvil.

🛡️ Firebase Integrado

Integración nativa directa con los servicios en la nube de Google Firebase para dar soporte robusto a proyectos listos para producción:

  • Hosting: Alojamiento estático ultrarrápido y seguro.
  • Auth: Autenticación nativa utilizando el proveedor Google Identity.
  • Realtime DB: Control de Whitelist de accesos y persistencia.
  • Analytics: Diccionario taxonómico unificado de telemetría.

⚙️ Pipeline con Gulp

Un potente compilador automatizado mediante scripts de consola. Asegura tu código de cliente y optimiza la velocidad antes de desplegar:

  • Minificación: Compresión al extremo de HTML, CSS y JS.
  • Ofuscación: Protección intelectual de la lógica del cliente.
  • Inyección: Reemplazo automático de variables de versión en HTML.
  • Distribución: Copiado y estructuración automática de assets y manifiestos.

🎮 Componentes Interactivos

1. Componente <bap-logo>

Muestra la identidad visual corporativa. Carga adaptativamente la constante de logo o utiliza un SVG premium vectorial.

<bap-logo></bap-logo>

2. Componente <bap-svg-image>

Gestor estandarizado de vectores SVG para iconos responsivos sin llamadas HTTP individuales.

<bap-svg-image name="chevronLeft" size="x3"></bap-svg-image>

3. Componente <bap-notification>

Notificaciones Toast y Snackbar con estilos inyectados directamente al Shadow Root para evitar fugas visuales.

import { bapNotify } from "./_main/util.js";
bapNotify(type, severity, message);

4. Componente <bap-spinner>

Indicador de carga animado, modular y estilizado con la paleta de colores del framework.

<bap-spinner></bap-spinner>

5. Componente <bap-dialog>

Modal premium flotante con glassmorphism, difuminado de fondo y enlaces asíncronos Base64.

<bap-dialog id="demo" title-main="...">
  Cuerpo del Diálogo
</bap-dialog>

6. Componente <bap-loading-state>

Bloqueador de pantalla completa para simular operaciones asíncronas largas y llamadas de red.

<bap-loading-state sub-message="...">
  Procesando...
</bap-loading-state>

💻 Ciclo de Desarrollo en la Terminal

$ npm run server

Servidor de Desarrollo

Levanta el servidor HTTP local con Hot-Reload y recarga en tiempo real sobre el navegador.

$ npm run optimize

Optimizar para Producción

Compila, comprime, aplica I18N y ofusca el código fuente moviéndolo a la carpeta /public.

$ npm run deploy-prod

Desplegar a Firebase

Sube y aloja instantáneamente tu aplicación optimizada en la infraestructura productiva de Firebase.
